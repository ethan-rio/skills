System topology & trust boundaries

Client Service Data store Infra —— sync · · · async / event

Trust boundaries

Three: client → edge (TLS termination), edge → VPC (mTLS via gateway), VPC → data tier (private subnets, IAM auth).

Hot path

Web → Cloudflare → API gateway → aurora-api → Postgres. 80% of read traffic served from Redis cache layer in front of Postgres reads.

Failure isolation

workers and events bus are async; outages there don't block the hot path. notes-svc has its own pool — CRDT spikes don't starve auth.
