# Productivity

General workflow tools for staying aligned with your agent before writing code.

All skills in this bucket are vendored from [`mattpocock/skills`](https://github.com/mattpocock/skills) (MIT). See top-level `LICENSE` for attribution.

- **[grill-me](./grill-me/SKILL.md)** — Get relentlessly interviewed about a plan or design until every branch of the decision tree is resolved.
- **[grill-with-docs](./grill-with-docs/SKILL.md)** — Grilling session that challenges your plan against the existing domain model and updates `CONTEXT.md` + ADRs inline.
- **[to-prd](./to-prd/SKILL.md)** — Turn the current conversation context into a PRD and publish it to the project issue tracker.
- **[write-a-skill](./write-a-skill/SKILL.md)** — Create new agent skills with proper structure, progressive disclosure, and bundled resources.
